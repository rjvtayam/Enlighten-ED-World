JavaScript Variables & Data Types

1. What are Variables?

Variables are containers for storing data values in JavaScript. They act like labeled boxes where you can store and manage different types of information.

// Variable declaration examples
let name = "John Doe";     // String variable
let age = 25;              // Number variable
let isStudent = true;      // Boolean variable

2. Declaring Variables

Using let and const

JavaScript provides two main ways to declare variables:

  • let: Allows reassignment of values
  • const: Creates a constant that cannot be reassigned
// let allows reassignment
let age = 25;
age = 26;  // This is allowed

// const prevents reassignment
const birthYear = 1998;
// birthYear = 1999;  // This would cause an error

3. Data Types in JavaScript

Primitive Data Types

String

Represents text data

let message = "Hello, World!";

Number

Numeric values (integers and decimals)

let price = 19.99;

Boolean

True or false values

let isLoggedIn = true;

Undefined

Variable declared but not assigned

let username;

Null

Intentional absence of value

let emptyValue = null;

4. Dynamic Typing

JavaScript automatically determines the type of a variable based on its value.

let age = 25;        // age is a number
age = "Twenty-five"; // Now age is a string
console.log(typeof age);  // Outputs: "string"

5. Type Coercion

JavaScript automatically converts data types when performing operations.

// String and number combination
let result = "5" + 10;  // Result is "510"
console.log(result);    // Outputs: "510"

// Numeric conversion
let sum = "10" - 5;    // Result is 5
console.log(sum);      // Outputs: 5
