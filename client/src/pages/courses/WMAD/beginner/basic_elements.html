Essential HTML Elements

Heading Elements


<h1>Main Heading (Most Important)</h1>
<h2>Section Heading</h2>
<h3>Subsection Heading</h3>
<h4>Sub-subsection Heading</h4>
<h5>Minor Heading</h5>
<h6>Least Important Heading</h6>

Headings create a hierarchical structure for content. H1 is the most important, while H6 is the least.

Best Practices:
  • Use only one H1 per page
  • Create a logical heading hierarchy
  • Help with SEO and accessibility

Text and Formatting Elements


<p>Regular paragraph text</p>

<strong>Bold text</strong>
<em>Emphasized (italic) text</em>
<u>Underlined text</u>

<mark>Highlighted text</mark>
<del>Deleted text</del>
<ins>Inserted text</ins>

<sup>Superscript</sup>
<sub>Subscript</sub>

Text formatting elements help emphasize and structure content within paragraphs.

Usage Tips:
  • Use semantic elements for meaning
  • Avoid overusing formatting
  • Prefer CSS for styling

Links and Navigation


<!-- External Link -->
<a href="https://example.com">
    External Website
</a>

<!-- Internal Page Link -->
<a href="#section">
    Jump to Section
</a>

<!-- Email Link -->
<a href="mailto:info@example.com">
    Send Email
</a>

<!-- Phone Link -->
<a href="tel:+1234567890">
    Call Us
</a>

Links connect web pages and provide navigation within and between websites.

Link Attributes:
  • href: Destination URL
  • target: Link behavior
  • title: Tooltip text

List Elements


<!-- Unordered List -->
<ul>
    <li>Bullet point 1</li>
    <li>Bullet point 2</li>
</ul>

<!-- Ordered List -->
<ol>
    <li>First item</li>
    <li>Second item</li>
</ol>

<!-- Definition List -->
<dl>
    <dt>Term</dt>
    <dd>Description</dd>
</dl>

Lists organize content in structured, easy-to-read formats.

List Variations:
  • Unordered (bullet) lists
  • Ordered (numbered) lists
  • Definition lists

Images and Multimedia


<!-- Basic Image -->
<img src="image.jpg" 
     alt="Description" 
     width="300" 
     height="200">

<!-- Image with Caption -->
<figure>
    <img src="photo.jpg" alt="Photo">
    <figcaption>Image Caption</figcaption>
</figure>

<!-- Multimedia Elements -->
<video controls>
    <source src="video.mp4" type="video/mp4">
</video>

<audio controls>
    <source src="audio.mp3" type="audio/mpeg">
</audio>

Multimedia elements allow embedding of images, videos, and audio in web pages.

Best Practices:
  • Always use alt text for images
  • Optimize media file sizes
  • Provide fallback content

Container and Semantic Elements


<!-- Generic Containers -->
<div>Block-level container</div>
<span>Inline container</span>

<!-- Semantic Structure -->
<header>Page or section header</header>
<nav>Navigation menu</nav>
<main>Main content area</main>
<article>Independent content</article>
<section>Thematic grouping</section>
<aside>Side content</aside>
<footer>Page or section footer</footer>

Semantic elements provide meaning to the structure of web content.

Semantic Benefits:
  • Improved accessibility
  • Better SEO
  • More meaningful code
