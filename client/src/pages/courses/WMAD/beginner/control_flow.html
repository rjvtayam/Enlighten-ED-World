Understanding Control Flow in JavaScript

What is Control Flow?

Control flow refers to the order in which statements are executed in a program. It allows developers to make decisions and control the execution path of their code.

Importance of Control Flow

  • Enables decision-making in programs
  • Allows conditional execution of code
  • Helps create dynamic and responsive applications
  • Provides flexibility in handling different scenarios

Conditional Statements

1. if Statement


let age = 18;
if (age >= 18) {
    console.log("You are an adult.");
}

Executes code only if a specified condition is true.

2. else Statement


let age = 16;
if (age >= 18) {
    console.log("You are an adult.");
} else {
    console.log("You are a minor.");
}

Provides an alternative block of code to execute when the if condition is false.

3. else if Statement


let age = 20;
if (age < 18) {
    console.log("You are a minor.");
} else if (age >= 18 && age < 60) {
    console.log("You are an adult.");
} else {
    console.log("You are a senior.");
}

Allows checking multiple conditions sequentially.

Comparison Operators

Operator Description Example
== Equality (loose comparison) 5 == "5" (true)
=== Strict Equality (type and value) 5 === "5" (false)
!= Inequality 5 != 6 (true)
> Greater than 10 > 5 (true)
< Less than 5 < 10 (true)
>= Greater than or equal to 10 >= 10 (true)
<= Less than or equal to 5 <= 10 (true)

Logical Operators

&&, ||, and ! Operators


let age = 25;
let hasTicket = true;

// AND (&&) operator
if (age >= 18 && hasTicket) {
    console.log("You can enter the event.");
}

// OR (||) operator
let isWeekend = true;
let isHoliday = false;
if (isWeekend || isHoliday) {
    console.log("Time to relax!");
}

// NOT (!) operator
let isLoggedIn = false;
if (!isLoggedIn) {
    console.log("Please log in.");
}

Switch Statement


let day = 2;
switch (day) {
    case 1:
        console.log("Monday");
        break;
    case 2:
        console.log("Tuesday");
        break;
    case 3:
        console.log("Wednesday");
        break;
    default:
        console.log("Invalid day");
}

Provides an alternative to multiple if...else conditions when checking one variable against different possible values.

Ternary Operator


let age = 18;
let result = age >= 18 ? "Adult" : "Minor";
console.log(result);  // Output: "Adult"

A shorthand version of the if...else statement for simple conditional assignments.

Truthy and Falsy Values

Falsy Values

  • false
  • 0 (zero)
  • "" (empty string)
  • null
  • undefined
  • NaN

Example


let name = "";
if (name) {
    console.log("Name exists.");
} else {
    console.log("Name is empty.");
}

Interactive Control Flow Demo

Age Verification

How This Works


function checkAge() {
    // Get the age input value
    const age = parseInt(ageInput.value);

    // Control Flow using if-else statements
    if (age < 0) {
        // First condition: Check for invalid age
        ageResult.textContent = "Invalid age! Age cannot be negative.";
        ageResult.style.color = "red";
    } else if (age < 18) {
        // Second condition: Check if user is a minor
        ageResult.textContent = "You are a minor.";
        ageResult.style.color = "blue";
    } else if (age >= 18 && age < 60) {
        // Third condition: Check if user is an adult
        // Using logical AND (&&) operator to combine conditions
        ageResult.textContent = "You are an adult.";
        ageResult.style.color = "green";
    } else {
        // Final condition: If none of the above conditions are true
        ageResult.textContent = "You are a senior.";
        ageResult.style.color = "purple";
    }
}

This demo demonstrates several key control flow concepts:

  • if-else Statements: Allows the program to execute different code blocks based on conditions
  • Logical Operators: Uses && (AND) to combine multiple conditions
  • Type Conversion: Uses parseInt() to convert input to a number
  • Conditional Branching: The code checks conditions in sequence and executes the first matching block

Day of Week

How This Works


function checkDay() {
    // Get the selected day value
    const day = daySelect.value;

    // Control Flow using switch statement
    switch (day) {
        case "1":
            // First case: Monday
            dayResult.textContent = "It's Monday! Start of the work week.";
            break;
        case "2":
            // Second case: Tuesday
            dayResult.textContent = "It's Tuesday! Keep pushing forward.";
            break;
        // ... other cases ...
        default:
            // Fallback case if no other cases match
            dayResult.textContent = "Invalid day selected.";
    }
}

This demo showcases the switch statement for control flow:

  • Switch Statement: Provides an alternative to multiple if-else conditions
  • Case Blocks: Each case represents a specific condition
  • Break Statement: Prevents fall-through to next case
  • Default Case: Handles scenarios not covered by specific cases
