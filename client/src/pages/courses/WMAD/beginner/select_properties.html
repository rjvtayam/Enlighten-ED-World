CSS Selectors & Properties

Lesson Overview

In this lesson, you'll learn about CSS selectors and how to apply styles to HTML elements.

CSS Selectors

Types of Selectors

1. Element Selector

Selects all elements of a specific type.


/* Selects all paragraph elements */
p {
    color: navy;
    font-size: 16px;
}

/* Selects all heading elements */
h1 {
    text-align: center;
    color: darkgreen;
}

Use Case: Apply styles to all elements of a specific type across your webpage.

2. Class Selector

Selects elements with a specific class attribute.


/* HTML */

Highlighted Text


Another highlighted element


/* CSS */
.highlight {
    background-color: yellow;
    font-weight: bold;
}

Use Case: Apply styles to multiple elements with the same class, regardless of their type.

3. ID Selector

Selects a single element with a specific ID attribute.


/* HTML */

Primary Header


/* CSS */
#main-header {
    color: crimson;
    font-size: 24px;
    text-transform: uppercase;
}

Use Case: Style a unique element on the page. Each ID should be unique.

4. Attribute Selector

Selects elements based on their attributes or attribute values.


/* Selects all inputs with type "text" */
input[type="text"] {
    border: 2px solid gray;
    padding: 5px;
}

/* Selects elements with a specific attribute */
[data-category] {
    background-color: lightblue;
}

/* Selects elements where attribute starts with a value */
a[href^="https"] {
    color: green;
    text-decoration: underline;
}

Use Case: Target elements with specific attributes or attribute patterns.

5. Pseudo-class Selector

Selects elements based on a specific state or condition.


/* Styles a link when hovered */
a:hover {
    color: red;
    text-decoration: underline;
}

/* Styles the first child element */
li:first-child {
    font-weight: bold;
}

/* Styles every odd row in a table */
tr:nth-child(odd) {
    background-color: #f2f2f2;
}

/* Styles an input when it's focused */
input:focus {
    outline: 2px solid blue;
    border-color: lightblue;
}

Use Case: Add interactive and dynamic styles based on element states.

Selector Combinations


/* Descendant Selector */
div p {
    color: purple;
}

/* Child Selector */
ul > li {
    list-style-type: square;
}

/* Adjacent Sibling Selector */
h1 + p {
    font-style: italic;
}

CSS Properties

Common CSS Properties

1. Color Property

Defines the text color of an element.


/* Named colors */
p { color: blue; }

/* Hexadecimal color */
h1 { color: #FF0000; }

/* RGB color */
div { color: rgb(0, 128, 255); }

/* RGBA with transparency */
span { color: rgba(0, 0, 0, 0.7); }
Key Points:
  • Can use named colors, hex, RGB, and RGBA
  • Affects text color of the element
  • RGBA allows for color transparency

2. Background Color

Sets the background color of an element.


/* Solid background color */
.box { background-color: lightgreen; }

/* Hexadecimal background */
.header { background-color: #F0F0F0; }

/* RGB background */
.section { background-color: rgb(200, 220, 255); }

/* Transparent background */
.overlay { background-color: rgba(0, 0, 0, 0.5); }
Key Points:
  • Fills the entire element with a specified color
  • Can create transparent overlays with RGBA
  • Useful for creating visual separation

3. Font Size

Determines the size of text within an element.


/* Absolute sizes */
body { font-size: 16px; }
h1 { font-size: 32px; }

/* Relative sizes */
p { font-size: 1em; }  /* Relative to parent element */
span { font-size: 1.5rem; }  /* Relative to root element */

/* Percentage sizes */
.small-text { font-size: 80%; }
.large-text { font-size: 120%; }
Key Points:
  • Supports pixels (px), em, rem, and percentage
  • em is relative to parent element size
  • rem is relative to root element size

4. Margin

Creates space outside an element, between the element and other elements.


/* Individual sides */
.box {
    margin-top: 10px;
    margin-right: 15px;
    margin-bottom: 10px;
    margin-left: 15px;
}

/* Shorthand: top, right, bottom, left */
.container {
    margin: 10px 20px 10px 20px;
}

/* Shorthand: vertical horizontal */
.element {
    margin: 15px 30px;
}

/* Centering with auto */
.center-block {
    margin: 0 auto;
    width: 50%;
}
Key Points:
  • Creates space outside the element
  • Can set individual or multiple sides
  • 'auto' can center block elements horizontally

5. Padding

Creates space inside an element, between content and border.


/* Individual sides */
.button {
    padding-top: 5px;
    padding-right: 10px;
    padding-bottom: 5px;
    padding-left: 10px;
}

/* Shorthand: top, right, bottom, left */
.card {
    padding: 10px 15px 10px 15px;
}

/* Shorthand: vertical horizontal */
.container {
    padding: 20px 30px;
}
Key Points:
  • Creates space inside the element
  • Increases the element's total size
  • Can set individual or multiple sides

6. Border

Defines the border around an element.


/* Full border specification */
.box {
    border-width: 2px;
    border-style: solid;
    border-color: black;
}

/* Shorthand */
.card {
    border: 1px solid #cccccc;
}

/* Individual border sides */
.element {
    border-top: 3px dashed red;
    border-bottom: 2px dotted blue;
}

/* Border radius for rounded corners */
.rounded {
    border: 1px solid gray;
    border-radius: 10px;
}
Key Points:
  • Combines width, style, and color
  • Can set individual border sides
  • border-radius creates rounded corners
