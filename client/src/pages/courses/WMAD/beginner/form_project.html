Interactive Form Mini Project

Mini Project: Creating an Interactive Form

Objective

Create an interactive form that validates user input, provides real-time feedback, and demonstrates basic form handling techniques.

Project Requirements

  1. Form Structure

    Create a form with the following fields:

    • Full Name (text input)
    • Email Address (email input)
    • Age (number input)
    • Password (password input)
    • Confirm Password (password input)
  2. Validation Requirements
    • Full Name: Required, minimum 2 characters
    • Email: Must be a valid email format
    • Age: Between 18 and 100
    • Password: Minimum 8 characters, must contain at least one uppercase, one lowercase, and one number
    • Confirm Password: Must match the password
  3. Interactive Features
    • Real-time validation as user types
    • Display error messages for invalid inputs
    • Show success message when all fields are valid

Bonus Challenges

  • Add a strength meter for the password
  • Implement additional custom validations
  • Add visual indicators for valid/invalid fields
