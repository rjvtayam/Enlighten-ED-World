What is HTML?

Definition

HTML (HyperText Markup Language) is the standard markup language for creating web pages. It describes the structure of a web page and its content.

Purpose

HTML provides the basic structure of sites, which is then enhanced and modified by other technologies like CSS and JavaScript.

History

Created by Tim Berners-Lee in 1989, HTML has evolved through multiple versions, with HTML5 being the current standard.

HTML Fundamentals

Basic HTML Structure


<!DOCTYPE html>
<html lang="en">
    <head>
        <meta charset="UTF-8">
        <title>Page Title</title>
    </head>
    <body>
        Page Content
    </body>
</html>

Core HTML Elements

  • <h1> to <h6>: Headings
  • <p>: Paragraphs
  • <a>: Hyperlinks
  • <img>: Images
  • <div>: Divisions
  • <span>: Inline container

HTML Attributes


<tag attribute="value">
    Content
</tag>

Examples:
<a href="https://example.com">Link</a>
<img src="image.jpg" alt="Description">
