Understanding the Box Model

Every HTML element is treated as a box with four key layers that control its layout and spacing.

1. Content

The actual content of the element (text, image, etc.)


.content-example {
    width: 200px;   /* Controls content width */
    height: 100px;  /* Controls content height */
}

2. Padding

Space between content and border


.padding-example {
    padding: 20px;  /* Adds space inside the border */
    padding-top: 10px;
    padding-right: 15px;
}

3. Border

Decorative line around the padding


.border-example {
    border: 2px solid #3498db;
    border-radius: 5px;
}

4. Margin

Space outside the border, between elements


.margin-example {
    margin: 20px;  /* Adds space outside the border */
    margin-bottom: 10px;
}

üìè Element Sizing and Spacing

Controlling Element Dimensions

Learn how to precisely control element sizes and spacing using CSS properties.

Box Sizing

Determines how the total width and height of an element are calculated


/* Content Box (Default) */
.content-box {
    box-sizing: content-box;
    width: 200px;  /* Does not include padding/border */
}

/* Border Box */
.border-box {
    box-sizing: border-box;
    width: 200px;  /* Includes padding and border */
}

Max and Min Dimensions

Set minimum and maximum sizes for responsive design


.responsive-element {
    min-width: 200px;
    max-width: 600px;
    min-height: 100px;
    max-height: 300px;
}

Modern CSS Layout Methods

Explore powerful layout techniques to create complex web designs.

Flexbox Layout

One-dimensional layout method for flexible element arrangement


.flex-container {
    display: flex;
    justify-content: space-between;
    align-items: center;
}

.flex-item {
    flex: 1;
    margin: 10px;
}

CSS Grid

Two-dimensional layout system for complex grid-based designs


.grid-container {
    display: grid;
    grid-template-columns: repeat(3, 1fr);
    gap: 20px;
}

.grid-item {
    background-color: #3498db;
}

üì± Responsive Design Principles

Creating Adaptive Web Layouts

Techniques to ensure your website looks great on all device sizes.

Media Queries

Apply different styles based on device characteristics


/* Mobile Devices */
@media screen and (max-width: 600px) {
    .responsive-element {
        width: 100%;
        flex-direction: column;
    }
}

/* Tablets */
@media screen and (min-width: 601px) and (max-width: 1024px) {
    .responsive-element {
        width: 90%;
        padding: 15px;
    }
}

Flexible Units

Use relative units for scalable designs


.responsive-container {
    width: 90%;     /* Percentage width */
    max-width: 1200px;
    margin: 0 auto;
}

.flexible-text {
    font-size: 1.5vw;  /* Viewport-relative sizing */
    padding: 1em;      /* Relative to font size */
}
