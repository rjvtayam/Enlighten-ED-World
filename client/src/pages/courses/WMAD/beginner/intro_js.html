Introduction to JavaScript

1. What is JavaScript?

JavaScript is a powerful programming language that makes websites interactive and dynamic. It allows you to create responsive web pages that can update and change without reloading the entire page.

2. Setting Up JavaScript

You can add JavaScript to an HTML page in three ways:

  • Inline: Using the <script> tag directly in HTML
  • Internal: Adding a <script> tag in the HTML file
  • External: Linking a separate .js file

3. Basic JavaScript Syntax

Learn how to declare variables and use different data types:


// Variable declarations
let name = "John";        // String
const age = 25;           // Number
let isStudent = true;     // Boolean

4. Operators in JavaScript

JavaScript supports various operators:


// Arithmetic Operators
let sum = 5 + 3;          // Addition
let difference = 10 - 4;  // Subtraction

// Comparison Operators
let isEqual = (5 === 5);  // Strict equality
let isGreater = (10 > 5); // Greater than

5. Simple Output with JavaScript

Display messages using different methods:


// Browser alert
alert("Hello, World!");

// Console output
console.log("Debugging message");

6. Basic JavaScript Functions

Functions help you organize and reuse code:


function greet(name) {
    return `Hello, ${name}!`;
}

console.log(greet("Alice")); // Outputs: Hello, Alice!

7. Conditional Statements

Make decisions in your code:


let number = 10;
if (number > 0) {
    console.log("Number is positive");
} else if (number < 0) {
    console.log("Number is negative");
} else {
    console.log("Number is zero");
}

8. Loops in JavaScript

Repeat actions multiple times:


// For loop
for (let i = 0; i < 5; i++) {
    console.log(i);
}

// While loop
let count = 0;
while (count < 3) {
    console.log(count);
    count++;
}

9. JavaScript Arrays

Store and manipulate collections of data:


let fruits = ["Apple", "Banana", "Orange"];
console.log(fruits[0]);    // Outputs: Apple

fruits.push("Mango");      // Add an element
fruits.pop();              // Remove the last element

Exercise 1: Age Checker

Exercise Goal

Create a program that checks if a user is old enough to vote based on their age.

Learning Objectives

  • Use conditional statements
  • Convert input to a number
  • Manipulate DOM elements

Code Explanation


// Function to check voting eligibility
function checkVotingEligibility() {
    // Get the age input element
    const ageInput = document.getElementById('ageInput');
    
    // Convert input to a number
    const age = parseInt(ageInput.value);
    
    // Use conditional statement to check age
    if (isNaN(age)) {
        // Handle invalid input
        ageResult.textContent = "Please enter a valid age.";
    } else if (age >= 18) {
        // Check if age is 18 or above
        ageResult.textContent = "You can vote!";
    } else {
        // Age is below 18
        ageResult.textContent = "You cannot vote yet.";
    }
}

Exercise 3: To-Do List

Exercise Goal

Create a simple To-Do List application with the ability to add and delete tasks.

Learning Objectives

  • Create and manipulate DOM elements
  • Handle user events
  • Implement basic list management

Code Explanation


// Function to add a new task to the list
function addTask() {
    // Get input and list elements
    const todoInput = document.getElementById('todoInput');
    const todoList = document.getElementById('todoList');
    
    // Check if input is not empty
    if (todoInput.value.trim() !== "") {
        // Create a new list item
        const li = document.createElement('li');
        li.textContent = todoInput.value;
        
        // Create a delete button for the task
        const deleteBtn = document.createElement('button');
        deleteBtn.textContent = 'Delete';
        deleteBtn.addEventListener('click', function() {
            // Remove the task when delete is clicked
            todoList.removeChild(li);
        });
        
        // Add delete button to the list item
        li.appendChild(deleteBtn);
        
        // Add list item to the todo list
        todoList.appendChild(li);
        
        // Clear the input field
        todoInput.value = "";
    }
}
