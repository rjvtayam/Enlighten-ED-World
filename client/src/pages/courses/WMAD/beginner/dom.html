What is the DOM?

Definition

The Document Object Model (DOM) represents the structure of an HTML document as a tree of objects.

Purpose

The DOM allows JavaScript to interact with HTML elements, enabling dynamic modifications such as:

  • Reading content
  • Changing styles
  • Adding/removing elements

DOM Manipulation Methods

1. Selecting Elements

document.getElementById()

let element = document.getElementById("myElement");
console.log(element);  // Logs the element with id="myElement"

document.querySelector()

let element = document.querySelector(".myClass");
console.log(element);  // Logs the first element with class="myClass"

document.querySelectorAll()

let elements = document.querySelectorAll("p");
console.log(elements);  // Logs all 
 elements on the page

2. Changing Content

innerHTML

let element = document.getElementById("myElement");
element.innerHTML = "Hello, World!";  // Changes content to "Hello, World!"

textContent

let element = document.getElementById("myElement");
element.textContent = "This is some text.";  // Changes text to "This is some text."

3. Changing Styles

let element = document.getElementById("myElement");
element.style.color = "blue";  // Changes text color to blue
element.style.backgroundColor = "yellow";  // Changes background color to yellow

4. Creating New Elements

let newElement = document.createElement("div");
newElement.textContent = "This is a new div!";
document.body.appendChild(newElement);  // Adds the new div to the page

5. Removing Elements

let element = document.getElementById("myElement");
element.remove();  // Removes the element with id="myElement"

6. Adding and Removing Classes

classList.add()

let element = document.getElementById("myElement");
element.classList.add("newClass");  // Adds class "newClass" to the element

classList.remove()

let element = document.getElementById("myElement");
element.classList.remove("oldClass");  // Removes class "oldClass"

7. Event Handling

let button = document.getElementById("myButton");
button.addEventListener("click", function() {
  alert("Button clicked!");
});

8. Traversing the DOM

let child = document.getElementById("myChild");
let parent = child.parentElement;  // Gets the parent of the element with id="myChild"
let children = parent.children;  // Gets all children of the parent element
