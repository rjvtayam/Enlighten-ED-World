What is CSS?

CSS Fundamentals

CSS (Cascading Style Sheets) is a styling language used to describe the presentation of a document written in HTML or XML.


/* Basic CSS Syntax */
selector {
    property: value;
}

/* Example */
body {
    font-family: Arial, sans-serif;
    background-color: #f4f4f4;
}

CSS Importance

  • Separates design from content
  • Enables consistent styling across pages
  • Improves website responsiveness
  • Enhances user experience

/* Styling Multiple Elements */
h1, h2, h3 {
    color: #333;
    text-align: center;
}

.button {
    background-color: #3498db;
    color: white;
    padding: 10px 20px;
}

CSS Integration Methods

Inline CSS


<h1 style="color: blue; 
           font-size: 24px;">
    Inline Styled Heading
</h1>

Applied directly to individual HTML elements

Internal CSS


<style>
    body {
        background-color: #f0f0f0;
    }
    p {
        color: green;
    }
</style>

Defined within the HTML document's <style> tag

External CSS


<link rel="stylesheet" 
      href="styles.css">

/* styles.css */
body {
    font-family: Arial, sans-serif;
}

Linked from a separate .css file

Advanced CSS Concepts

CSS Color Modes


/* Color Representations */
.text-hex { color: #3498db; }
.text-rgb { color: rgb(52, 152, 219); }
.text-rgba { color: rgba(52, 152, 219, 0.5); }
.text-hsl { color: hsl(204, 70%, 53%); }

Explore different ways to define colors in CSS

CSS Units


/* Absolute Units */
div { width: 200px; }
body { font-size: 16pt; }

/* Relative Units */
.container { width: 50%; }
.text { font-size: 1.5em; }
.viewport { height: 100vh; }

Understanding absolute and relative measurement units

CSS Flexbox


.flex-container {
    display: flex;
    justify-content: center;
    align-items: center;
    gap: 10px;
}

.flex-item {
    flex: 1;
    background-color: #3498db;
